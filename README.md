# Blog Preview Card

## 📌 Overview
This project is a solution to the Frontend Mentor Blog Preview Card challenge.  
The goal was to build a clean and responsive blog card layout.

- HTML5
- CSS3
- Flexbox
- Mobile-first workflow

## 📱 Features
- Responsive design (mobile & desktop)
- Clean card layout
- Styled typography
- Hover effects

## 📚 What I learned
- How to build card components
- Working with spacing and layout
- Improving CSS styling and structure
- Using hover effects for better UI
